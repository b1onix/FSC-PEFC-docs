```
Requirement(s) Clause 1.3, Clause 5.5
```
Publication date 14 May 2018
Is it allowed to omit the license code when using the name ‘Forest Stewardship Council’ or the
initials ‘FSC’ in social media posts?
Given the particularities of this communication medium, it is allowed for a certificate holder to omit the
license code in a social media post, so long as either the social media account or the post itself
references the certificate holder’s website where the compulsory promotional elements are listed.

```
Is it allowed for the certification body to approve generic formats (templates) for their clients?
```
In the absence of an approved trademark use management system, certificate holders shall submit all
intended uses of FSC trademarks to the certification body for approval. There are however situations
```
when the certification body is able to approve a generic format (template) to be used on a range of
```
```
products to which different designs are being applied (e.g. a postcard with different images). This is
```
allowed when the following conditions are met:
1. The FSC label is used on the same product type without any alterations (e.g. changing the label
```
title, changing the product type, changing the size of the label etc.).
```

When a printed publication is FSC labeled, is it allowed to print calculations on savings on

Code INT-STD-50-001_02
```
Requirement (s) Clause 3.7
```
Publication date 01 March 2011
Is it possible to add “contents’’ to the list of words for product types?
No, it is not allowed to use “contents” as a product type.
The product types are very general in nature, more of general material types actually, such as paper,
wood, board, rubber etc. For the sake of clarity there are a few real product names included, such as
packaging. The intention is to keep the list at a very general level, so it is not allowed to use “contents”
as a product type.

```
Requirement(s) Clause 4.3, Clause 7.1
```
Publication date 11 June 2025
Can the FSC trademarks be used next to other certification scheme marks?
Yes, the general rule is that FSC trademarks can be used next to other certification scheme marks.
However, there are restrictions to using FSC trademarks next to marks of other forest certification
schemes specifically:
```
1) On-product use of other forest certification scheme marks is not permitted if the product is FSC
```
```
labelled. (Clause 4.3)
```
```
2) Off-product use of other forest certification scheme marks is permitted only if such use does not
```
```
imply equivalence to FSC. (Clause 7.1)
```

Code INT-STD-50-001_31
```
Requirement(s) Clause 8.3
```
Publication date 14.08.2024
FSC-labelled products often include a barcode, which is a method of representing data in a
visual, machine-readable form, for automatic identification and data capture. Most commonly,
the barcode is printed on the product in black and white and standard FSC colours are not
otherwise available in the print area. Is it possible to apply clause 8.3 in this case?
Yes, clause 8.3 applies in this case. If standard colours are not available for the print area, except for
the black and white of the barcode, an available colour providing legible contrast on a solid background
may be used instead.
